# POS Point of Sales Website

Frontend-only POS business website built with HTML, CSS, and JavaScript.

## Files

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`

## GitHub Pages

1. Create a new GitHub repository.
2. Upload these files into the repository root.
3. Go to `Settings > Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Choose `main` and `/ (root)`.
6. Save.

If you want to use a custom domain, add a `CNAME` file with only the domain name inside, for example:

```txt
clainerdev.com
```

Then add the same domain in `Settings > Pages > Custom domain`.
